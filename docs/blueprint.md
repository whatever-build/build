# **App Name**: NeuroWallet AI

## Core Features:

- Secure Mnemonic Generation: Utilize an AI-powered tool for generating secure BIP39 mnemonics for digital asset recovery processes.
- Multi-Chain Wallet Interrogation: Scan and interrogate wallets across diverse blockchain networks (BTC, ETH, SOL, TRX, etc.) to identify recoverable digital assets.
- Heuristic Recovery Filter: Implement a probability-based AI tool to filter and prioritize mnemonics by checksum patterns, enhancing the efficiency of asset recovery.
- Robust API Management: Intelligently manage cycling through various blockchain RPCs (Ankr, Blockcypher, Public RPCs) with back-off logic for resilient and uninterrupted interrogation.
- Live Activity Terminal Feed: Display real-time processing statuses and interrogation results in a monospace, terminal-style feed with green/cyan text accents.

## Style Guidelines:

- Primary color: A vibrant violet (#AD4FE6) serves as the primary color, setting an energetic and modern tone reflective of the app's high-performance capabilities. This color is derived from the user's requested `hsl(263, 90%, 68%)`.
- Background color: A deep, almost black background (#0A0A0A) creates a sense of depth and allows vibrant foreground elements to stand out, evoking the dark interfaces common in high-tech systems. This background color is directly specified by the user.
- Accent color: A rich, deep indigo (#2937A3) provides a strong visual contrast, drawing attention to key interactive elements and important data without overpowering the primary violet. This analogous hue reinforces the sophisticated digital aesthetic.
- Headline and body text font: 'Inter' (sans-serif) for a modern, objective, and highly legible appearance across all main interface elements. Terminal feed font: 'Source Code Pro' (monospace) for clear, code-like display within the live activity terminal.
- Sleek, modern, and conceptually abstract icons that align with the 'Neural Engine Branding' and high-tech crypto themes, integrating seamlessly with ShadCN UI components.
- A modular and information-dense layout utilizing ShadCN UI components. Key interactive elements, such as active cards, will feature a 'snake border' hardware-accelerated conic-gradient to highlight their status and activity. The terminal feed will use a dedicated section with hidden scrollbars for a clean, immersive experience.
- Subtle, performant animations will be used for state changes, transitions, and to draw attention to active processes, such as the specified spinning 'snake border' around active cards and interactive particle backgrounds for login and dashboard.
